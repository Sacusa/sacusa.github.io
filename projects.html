    Projects

    NUP

    A Processor on FPGA

    • ​NUP is an 8-bit single-bus architecture, originally designed by Prof. R.N. Biswas. The processor was synthesized on Xilinx Spartan-6 FPGA.

    • A pedagogy focused design, it features a micro-coded control unit and a hardware stack with dedicated instructions for stack manipulation.

    • The original design was further enhanced by the inclusion of memory-mapped I/O and a stepper motor peripheral.

    • The project was mentored by Prof. Sanjay Gupta and has served the basis for further research in microprocessor design in our University.

    RTOS

    A RTOS for Fitness Devices

    Buit a real-time operating system (RTOS) for a Texas Instruments TM4C123 MCU powered fitness device. The RTOS included

    • a Round Robin pre-emptive scheduler, including blocking and sleeping support for threads

    • a mailbox interface and FIFO queues for IPC

    • a FAT-like filesystem for logging system data

    The project was a part of the labs in the MOOC ”Realtime Bluetooth Networks”. Project source can be found here.

    Jack

    A Complete Software Stack

    Wrote the entire Jack software stack for Hack CPU. The stack comprises of

    • An assembler, responsible for linking and converting assembly code into binary code. This was written in C.

    • A VM-based two-tier compiler for Jack, a Java-like programming language. The Jack-to-VM compiler was written in Python, while the VM-to-Assembly translator was written in Java.

    • Jack OS, a simple OS which provides support for additional data types (like String), memory management and drivers for a keyboard and a screen. This was written in Jack.

    Project source can be found here.
